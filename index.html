About AI Recipe Maker

AI Recipe Maker is a web application that helps users create and manage recipes using AI-powered chat. It features user authentication, chat history, and recipe suggestions.

Frontend

  • index.html: Main landing page with login and signup options, and a scroll-down button to view project info.
  • style2.css: Modern responsive styles for layout, navigation, container, info section, buttons, and forms.
  • script2.js: Handles login, signup, encryption, and scroll-down functionality. Communicates with the backend for authentication and registration.
  • script.js: Manages chat interactions, message display, chat history, deletion, and logout. Uses sessionStorage for authentication key and communicates with backend endpoints.

Backend

  • index.js: Node.js Express server with PostgreSQL integration. Handles user authentication, registration, chat message processing, chat history, and deletion. Integrates with OpenRouter AI for recipe suggestions.
  • Endpoints:
    • /signup: Registers new users (encrypted email/password).
    • /login: Authenticates users and returns success status.
    • /chat: Processes user messages, fetches AI response, and stores chat history.
    • /history: Returns chat history for authenticated users.
    • /delete: Deletes chat history for authenticated users.
  • Uses custom encryption/decryption for user credentials.
  • Stores users and messages in PostgreSQL database.

Features

  • Modern UI with responsive design and smooth scrolling.
  • Secure login and signup with encrypted credentials.
  • AI-powered chat for recipe creation and suggestions.
  • Chat history management (view and delete).
  • Session-based authentication using encrypted keys.

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express, PostgreSQL
  • AI Integration: OpenRouter API

How to Use

  1. Sign up or log in with your email and password.
  2. Start chatting with the AI to create or get recipe suggestions.
  3. View your chat history or delete it as needed.
  4. Scroll down for more information about the project.

Created by using modern web technologies and AI integration for a seamless recipe creation experience.
